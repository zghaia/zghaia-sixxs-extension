我真想说更新这个后不再更新了，但bug无处不在，需使用一段时间才能发现。

```

# v1.4 #
```
完善了网站过滤处理，

地址栏6调出的sissx，可以直接只输入bing之类的一级域名，直接默认当成是http://bing.com来打开。
这样加快了输入。当然如果根域名不是.com，则可以输入xxxx.net之类的，也是可以的。
默认还是可处理完整的url地址。

还有就是没开启sixxs时，地址栏6调用sixxs和右键链接sixxs功能依然可用，方便临时需要。
```

# v1.3 #
```
加入过滤网站设置选项，这样适用性就更大了，用得是一级域名来过滤，
如www.google.com就取出一级域名google来过滤，

如须增加其它网站的过滤，则在后面加分号 ;（英文输入法下），
然后后加上需要过滤网站的一级域名，如baidu，附在后面
就变成xxxxx;baidu
注意最后面一定不能有分号;
```

# v1.2 #
```
支持google搜索引擎了，但在搜索页面，不能直接点击页面来进入
须得右键链接，点击右键菜单里的sixxs项即可。
部分页面下，直接点击或者鼠标中键，无法获取url，但可以通过右键链接点击sixxs来打开。

还支持地址栏输入URL地址。可以是是
完整的http://\* 
或者缺少http://
之类的话，默认自动加上。 
不过不是直接在地址栏输入，enter就可以用，须的先键盘按数字6关键字，然后tab键，调出Sixxs
```

# v1.1 #
```
从书签里进入，须得是鼠标中键而不是直接点击鼠标左键来进入。当然右键书签新标签页打开也是可以的.
支持bing和baidu搜索引擎主页，从书签里鼠标中键进入，搜索后点击链接即可.
这样的话，若发现也页面直接点击打不开，
可以尝试右键新标签页打开，或者右键sixxs打开（google相关页面时用到）
以上是内网下的情况
如果有外网则地址栏输入或粘贴地址也可以
```

但右键sixxs和地址栏sixxs依然可用
```
